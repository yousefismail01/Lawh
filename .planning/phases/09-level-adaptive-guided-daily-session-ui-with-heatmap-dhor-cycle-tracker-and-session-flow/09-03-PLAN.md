**HeatmapGrid.tsx:**
A 30-column x 20-row SVG grid representing 604 mushaf pages. Props:
```typescript
interface HeatmapGridProps {
  memorizedJuz: number[]            // juz numbers that are memorized
  qualityScores: Record<number, number>  // juz -> quality 1-5
  isDark: boolean
}
```

Implementation:
- Use `react-native-svg` (Svg, Rect) — already installed
- Constants: CELL_SIZE = 9, GAP = 1, COLS = 30, ROWS = 20
- Total width: 30 * (9+1) = 300, height: 20 * (9+1) = 200
- Wrap in a View with horizontal centering
- For each juz (1-30) column, each page (1-20) row:
  - If juz is not in memorizedJuz set: fill with gray (#333 dark, #ddd light)
  - If memorized: color by quality score using getQualityColor helper:
    - quality >= 4.0: green (#34C759 light, #30D158 dark)
    - quality >= 3.0: yellow (#FFCC00 light, #FFD60A dark)
    - quality >= 2.0: orange (#FF9500 light, #FF9F0A dark)
    - quality < 2.0: red (#FF3B30 light, #FF453A dark)
  - Each cell: `<Rect x={col*(CELL_SIZE+GAP)} y={row*(CELL_SIZE+GAP)} width={CELL_SIZE} height={CELL_SIZE} fill={color} rx={1.5} />`
- Performance: useMemo for the cells array, keyed on memorizedJuz.length + JSON.stringify(qualityScores). This avoids re-rendering 600 rects on every render.
- Below the grid, show a color legend: 5 small colored squares with labels (Strong/Good/Moderate/Weak/Not memorized)
- Add a title row above: "Mushaf Heatmap" in fontWeight 600, fontSize 15
- Juz labels: show "1" and "30" at column edges below the grid, fontSize 10, muted color

- Stats row at bottom: "{reviewed} of {total} juz reviewed this cycle" + "~{remainingDays} days remaining"
- Card style: borderRadius 16, borderWidth hairline, padding 16 — match TodaySession card pattern
