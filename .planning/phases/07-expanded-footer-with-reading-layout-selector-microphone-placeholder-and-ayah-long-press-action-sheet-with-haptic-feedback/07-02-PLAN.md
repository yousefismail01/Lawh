- Import `* as Haptics from 'expo-haptics'`
- Fire `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Medium)` immediately when the long-press handler fires, BEFORE the async `quranService.getAyahText()` call. This gives instant feedback that the gesture registered.

2. In `AyahActionSheet.tsx`, add light haptic to ALL action button taps:
- Import `* as Haptics from 'expo-haptics'`
- Create a helper wrapper:
```typescript
const withHaptic = useCallback((action: () => void | Promise<void>) => {
  return async () => {
    await Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light);
    action();
  };
}, []);
```
- Wrap each Pressable's `onPress` with the haptic helper:
  - Bookmark: `onPress={withHaptic(handleBookmark)}`
  - Translation: `onPress={withHaptic(handleTranslation)}`
  - Tafsir: `onPress={withHaptic(handleTafsir)}`
