  <what-built>Complete Phase 7: expanded footer toolbar, reading mode selector, mic placeholder, haptic feedback, and audio playback in action sheet</what-built>
  <how-to-verify>
1. **Footer layout** — Open the mushaf. Tap to show chrome. Verify:
   - Footer has frosted glass blur background
   - Top row: layout icon (left), page number (center), mic button with accent color (right)
   - Bottom row: page slider (works as before — drag to navigate)
   - Footer auto-hides after 5 seconds

2. **Mic placeholder** — Tap the mic button. Verify:
   - "Recitation coming soon" toast appears
   - Toast auto-dismisses after ~2 seconds
   - Button looks inviting (accent colored, not grayed out)

3. **Layout selector** — Tap the layout icon. Verify:
   - Popover/sheet appears with 3 mode options (Mushaf, Arabic, Translation) and Tajweed toggle
   - Footer does NOT auto-hide while popover is open
   - Current mode is visually highlighted

4. **Arabic Cards mode** — Select "Arabic" from the layout selector. Verify:
   - View transforms to a scrollable list of ayah cards
   - Cards start at the same reading position (same page's ayahs)
   - Arabic text renders RTL with correct font
   - Scrolling works smoothly

5. **Translation Cards mode** — Open layout selector, select "Translation". Verify:
   - Cards show Arabic text + English translation (Sahih International)
   - No `[object Object]` or broken footnote markers in translation text
   - Text is readable and properly formatted

6. **Position sync** — Navigate to a card deep in the list (e.g. Surah Al-Imran). Switch back to Mushaf mode. Verify:
   - Mushaf opens near Al-Imran's pages (not back at Al-Fatiha)

7. **Tajweed toggle** — Toggle tajweed off in the layout selector. Switch to mushaf mode. Verify:
   - Text renders without tajweed colors (all same color)
   - Toggle on: tajweed colors return

8. **Haptic feedback** — Long-press an ayah word. Verify:
   - Medium haptic felt on the long-press moment
   - Action sheet slides up
   - Tapping any action button (Bookmark, Translation, Tafsir, Play Audio) triggers a light haptic

9. **Audio playback** — Long-press an ayah, tap "Play Audio". Verify:
   - Inline audio player appears in the action sheet
   - Audio plays the correct ayah recitation (Alafasy voice)
   - Play/pause works
   - Close the action sheet — audio stops

10. **Dark mode** — Switch device to dark mode. Verify footer, cards, popover, and action sheet all render with dark theme.
