  <action>
    1. Create `components/hifz/SurahCard.tsx`:
       - Props: `surahNumber: number`, `status: HifzStatus`, `onPress: () => void`
       - Compact card showing surah number centered
       - Background color per status using `buildColors(isDark)` pattern:
         - not_started: isDark ? '#2a2a2a' : '#f0f0f0' (neutral gray)
         - in_progress: isDark ? '#1a3a5c' : '#dbeafe' (blue tint)
         - memorized: isDark ? '#1a4a2a' : '#d1fae5' (green solid)
         - needs_review: isDark ? '#5c3a1a' : '#fef3c7' (amber/warning)
       - `React.memo` wrapped
       - Fixed size for 5-column grid (calculate from screen width minus padding)

    2. Create `components/hifz/SurahGrid.tsx`:
       - Uses `FlatList` with `numColumns={5}` for 5-column grid
       - Gets surah metadata from `quranService.getAllSurahs()` (114 surahs)
       - Gets statuses from `useHifzStore(state => state.surahStatuses)`
       - Calls `loadProgress('hafs')` on mount via `useEffect` if not loaded
       - Derives status per surah: if no progress entry -> 'not_started'; if any memorized + any needs_review -> 'needs_review' (review takes priority); if all memorized -> 'memorized'; otherwise 'in_progress'
       - On card press: opens SurahDetailSheet for that surah
       - `keyExtractor` by surah id

    3. Create `components/hifz/StatsPanel.tsx`:
       - Shows at top of Hifz tab above the grid
       - Reads from useHifzStore: totalMemorized, surahStatuses
       - Calls hifzService.getJuzStats('hafs') for juz breakdown
       - Display: "X / 6,236 ayahs memorized" with percentage
       - Strongest juz and weakest juz (by name: "Juz 30" etc.)
       - Average strength as a percentage
       - Uses buildColors pattern for dark mode
       - Compact horizontal layout fitting above the grid

    4. Create `components/hifz/AyahProgressRow.tsx`:
       - Single row for per-ayah display in the detail sheet
       - Props: ayahNumber, strengthScore, lastReviewed (date string or null), nextDue (date string or null), mistakeCount
       - Strength bar: colored View with width proportional to strength (0-100%)
       - Bar color: red (0-30%), orange (30-60%), yellow (60-80%), green (80-100%)
       - Shows "Last: Mar 5" and "Next: Mar 8" formatted dates, or "--" if null
       - React.memo wrapped

    5. Create `components/hifz/SurahDetailSheet.tsx`:
       - Half-height Modal bottom sheet (follows AyahActionSheet pattern)
       - Props: visible, surahId (number | null), onClose
       - Header: surah name (Arabic + transliteration from quranService), overall % memorized
       - Scrollable list of AyahProgressRow components
       - Loads ayah progress via `hifzService.getAyahProgress(surahId, 'hafs')` when visible
       - Action buttons at bottom:
         - "Start Review" -- navigates to review screen (router.push)
         - "Mark All Memorized" -- marks all ayahs in surah as memorized via hifzStore.markMemorized in a loop
       - Dark mode via useResolvedTheme + buildColors
  </action>
